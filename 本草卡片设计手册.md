## 本草卡片 — 产品设计手册

本草卡片是一款面向中医执业医师资格考试备考者的中药学习工具，以单文件 HTML 应用的形式交付，收录184味考纲要求掌握的中药材数据，支持翻转卡片浏览、默写练习、进度追踪三大核心功能。当前版本为 v1.2.0。


### 一、产品定位与使用场景

本产品专为2026年中医执业医师资格考试中药学科目设计。用户可在手机浏览器或桌面浏览器中打开单个 HTML 文件即可使用，无需安装、无需联网（字体加载除外）。核心使用场景包括：日常浏览记忆药材功效主治、针对性默写练习检验掌握程度、追踪学习进度分析薄弱环节。


### 二、信息架构：三标签布局

应用采用三标签导航结构，底部统计栏常驻显示当前筛选范围的掌握进度。

**浏览** — 翻转卡片式学习，适合初次接触和反复记忆。用户看到药材名和功效概要，点击卡片翻转查看完整的功效、主治、用法信息，通过"掌握"和"不熟"按钮自评掌握程度。

**练习** — 默写测试模式，提供三种子模式和灵活的范围筛选。用户根据提示主动回忆药材信息，是检验和巩固记忆的核心模块。

**进度** — 数据仪表盘，展示学习统计、日历打卡、分类完成度和薄弱药材列表，支持进度导出/导入/重置。


### 三、浏览模式

浏览模式是默认打开的页面，采用3D翻转卡片设计。

**卡片正面**显示药材所属分类（如"解表药 · 发散风寒药"）、药材名（大字居中显示）和功效概要。底部提示"轻点翻转"引导用户操作。

**卡片背面**展示完整信息：药材名、分类标签、用法用量（如有）、功效详文和主治内容。功效和主治分别以绿色和蓝色标签区分，形成视觉层次。

**底部导航**包含四个按钮：上一张、不熟（红色）、掌握（绿色）、下一张。点击"掌握"将该药材记入已掌握集合，点击"不熟"记入待巩固集合，两者均自动跳转至下一张卡片。在"薄弱"范围下标记掌握后，药材会从列表中移除，索引自动调整防止越界。

**分类筛选**通过顶部水平滚动的药丸按钮实现，包含"全部"和20个药材分类（解表药、清热药、泻下药等），点击切换后索引重置为0。


### 四、练习模式

练习模式是本应用的核心学习模块，提供子模式切换和范围筛选两层控制。

#### 4.1 子模式

**写药名** — 卡片展示药材的功效和主治信息，用户在输入框中输入药材名，点击"确认"或按回车键提交。答案正确显示绿色"回答正确"，错误显示红色"答案：XX"。每次作答后可展开查看用法用量。

**写功效** — 卡片展示药材名和分类，用户先在脑中回忆该药的功效主治，然后点击"点击查看答案"展开完整信息对照。展开后出现"没记住"（红色）和"记住了"（绿色）两个自评按钮，用户根据自身判断诚实评价，系统据此更新掌握状态并跳转下一张。

**随机** — 混合模式，根据药材ID的奇偶性自动分配题型（写药名或写功效）。同一药材每次进入始终是同一题型，确保学习体验一致。卡片顶部显示题型标签（绿色"写药名"或蓝色"写功效"）以便区分。

子模式通过练习页顶部的三个切换按钮快速切换，切换时重置当前作答状态但不改变范围筛选和索引位置。

#### 4.2 范围筛选

练习模式的分类筛选条比浏览模式多一个"薄弱"选项，排列顺序为：全部、薄弱、各药材分类。

"薄弱"按钮显示当前薄弱药材数量角标（如"薄弱 (12)"），选中后仅展示被标记为"不熟"的药材。当薄弱列表为空时，显示空状态提示"暂无薄弱药材，在浏览中标记'不熟'即可添加"。

选择不同范围时，索引重置为0，作答状态清空。


### 五、进度仪表盘

进度标签页提供全面的学习数据分析，从上到下包含以下模块：

**连续打卡卡片** — 三张统计卡分别显示连续打卡天数、累计学习天数和总体完成百分比。连续打卡在连续两天学习时递增，中断后从1重新开始。

**学习日历** — 35天（5周）网格日历，以周一为起始列。浅色背景表示当天学习了1-4味药材，深色表示5味及以上。当天日期以红色边框标记。底部图例说明颜色含义。

**分类统计** — 每个药材分类的进度条，显示已掌握数/总数。进度条长度按掌握比例填充，未学习的分类以低透明度显示。

**薄弱药材列表** — 以红色药丸形式展示所有标记为"不熟"的药材名，方便快速识别需要重点复习的品种。

**操作按钮** — 导出进度（下载 JSON 文件备份）、导入进度（从 JSON 文件恢复）、重置进度（清空所有学习数据，需确认）。


### 六、数据模型

应用内嵌184味中药材数据，每味药材包含以下字段：

- `id` — 唯一编号（1-184）
- `name` — 药材名
- `cat` — 大类（如"解表药"、"清热药"、"补虚药"等，共20类）
- `sub` — 小类（如"发散风寒药"、"清热泻火药"、"补气药"等）
- `effect` — 功效文本
- `indication` — 主治文本
- `dose` — 用法用量及注意事项（可为空）


### 七、状态管理与持久化

应用使用 `localStorage` 自动保存学习进度，存储键为 `bencao_v2`，数据结构为：

```json
{
  "kn": [1, 3, 7],       // 已掌握的药材ID列表
  "uk": [2, 5, 12],      // 待巩固（薄弱）的药材ID列表
  "days": {               // 每日学习记录
    "2026-06-15": {"n": 12}
  },
  "streak": 5,            // 连续打卡天数
  "last": "2026-06-15"    // 最后学习日期
}
```

每次标记"掌握"、"不熟"、答题正确/错误或自评时自动触发保存。进度导出功能将上述数据序列化为 JSON 文件下载，导入功能反向解析并覆盖当前状态。


### 八、交互设计

**翻转卡片** — 浏览模式下点击卡片任意位置触发3D翻转动画（CSS `rotateY(180deg)`），再次点击翻回。

**触控滑动** — 在卡片区域内左右滑动可切换上/下一张，滑动距离超过60像素且水平位移大于垂直位移的1.5倍时触发。

**键盘快捷键** — 左右方向键切换卡片，空格键下一张，回车键翻转（仅浏览模式）。在练习模式的输入框内，方向键和空格不触发导航以避免干扰输入。

**输入体验** — 练习模式的药名输入框自动获取焦点，支持回车键提交。输入过程中实时清空上次判定结果，确保每次作答独立。


### 九、视觉设计规范

**色彩体系** — 以古籍纸张为灵感的暖色调。主背景 `#f7f3eb`，卡片背景 `#faf8f3`，朱砂红 `#b5453a` 用于标题和操作强调，翡翠绿 `#4a8c6f` 用于掌握/正确状态，琥珀色 `#c9943a` 用于用法用量信息，天青色 `#5a8db5` 用于主治标签。暗色模式通过 `prefers-color-scheme: dark` 自动切换。

**字体方案** — 正文使用"霞鹜文楷"（LXGW WenKai），药材名使用"思源宋体"（Noto Serif SC）900字重突出显示。两者均从 Google Fonts CDN 加载。

**响应式** — 最大宽度520px居中布局，卡片区域 `min-height: 240px`，横屏（高度小于600px）时压缩为170px。底部按钮最小高度46px确保触控友好。

**动效** — 卡片翻转0.5秒贝塞尔曲线，切换滑入0.28秒，淡入0.35秒。所有交互按钮按下时有 `scale(0.96)` 的微缩反馈。
